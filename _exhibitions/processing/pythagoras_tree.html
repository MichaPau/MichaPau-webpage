Pythagoras tree

Angles are calculated with Perlin noise.
What gives the animation some kind of coral/plant hyprid impression.
Move the mouse to change the color hues..

I am using the p5.js library here. Processing.js was too slow for some reason.
Downside for this sketch is that p5.js doesn't support gradient fills on Vertex points. So the squares are only single color.
