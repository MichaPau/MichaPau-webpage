This is a list of some my examples from the EdX HTML5 course I took some time ago.

Some of them will be integrated in the upcoming version of this course on EdX. Be aware that some examples are prototypes I made some time ago. It may not always reflect absolute best practice.

HTML5 Video API

Video Player interaction

This is an example which shows some interaction on the timerupdate event in the form of speech bubbles which comment a movie trailer. It has some custom video controls in form of a remote control and other features for the video player.

See the video play in action (opens in a new tab).

Video chapter navigation

This example uses a vtt track to navigate in a video. This allows to navigate the video in a chapter like manner. In the moment there is a problem with firefox due the unsupported cuechange event.

See the video play in action (opens in a new tab).

Canvas Selfi Application

This is a selfi application. It uses the camera, the canvas, WebGL for rendering and pixi.js for the filters. It has 2 modes - normal and rendered in a polaroid frame.
In the moment this works only in Firefox because:
1. Chrome and Opera doesn't allow getUserMedia anymore without a SSL connection. (I won't purchase a certificate only for this example).
2. Internet Explorer doesn't support getUserMedia at all.

(1) will just fail silently for the user (error shows only in the console).

Use the selfi app (opens in a new tab)

Canvas Ball Game

Simple ball game rendered on the canvas. This was a playground for canvas animation, collision detection etc. Surly it's not an addictive great game but the goal was to play around with that stuff and make something actually playable. That was as well a possibility to use sound samples and try out the HTML5 Audio API.

Play the canvas game (opens in a new tab)
